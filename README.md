# Sniffer

Neste projeto o objetivo é aprender mais sobre a integração de Java com linguagens de mais baixo nível, como C. O recurso que permite integração e qua foi utilizado aqui é o JNI (Java Native Interface). Um Sniffer implementado em C (que utiliza basicamente RAW sockets) será integrado com uma GUI (Graphical User Interface). A ideia inicial é realizar uma simples integração e com o tempo novas features serão adicionadas.
